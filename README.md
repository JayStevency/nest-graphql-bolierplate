# NestJs + Graphql 

This is a project for bolierplate about nestjs and graphql.


## Business Logic

- Signup
- Login/Logout
- ContactUs

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
